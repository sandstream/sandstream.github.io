Nio byggen. En metod.

Projekt som visar hur Peter jobbar: tätt team, snabba beslut, idéer varje dag och stark dragning mot att leverera något verkligt på veckor, inte planera det i kvartal.

  • HunterMatcher — AI-driven konsultmatchning, från idé till produktion på tre månader. Mejlintag, AI-tolkning av uppdragskrav, vektorrepresentationer av konsultprofiler och realtidsrankning med Vertex AI. 180 konsulter matchas automatiskt. huntermatcher.frontwalker.se
  • Kjorre — kassasystem för föreningskiosker: QR-baserad självbetjäning, Swish- och Stripe-integration, kassaregister med journalkrav och adminportal per förening. Multi-tenant från dag ett. kjorre.com
  • Game Day DJ — realtidsljud för amatörsport. iOS-app med nedladdningsbara ljudpaket, fungerar utan uppkoppling. Live i App Store. gamedaydj.app
  • Ralph Inferno — AI-agnostisk orkestreringsmotor för autonom kodgenerering med explicita verifikationspunkter och full spårbarhet. Öppen källkod. github.com/sandstream/ralph-inferno
  • Lumen — automatiserad stiftelserapportering: FastAPI, React och integrationer mot Visma.NET ERP och Google Document AI. Produktionsklar plattform i drift.
  • Brand Brain — AI-driven kreativ pipeline: en RAG-baserad assistent som bär varumärkets tonalitet, tillåtna claims och visuella regler och genererar brandkonsekventa annonsvarianter. Python, Imagen 4, Claude, DeepL.
  • kit — säkerhets-CLI för AI-agenter och människor: från git clone till körbar miljö med ett kommando. 12 secret-backends, supply-chain-scanning, noll LLM-anrop, lokal-först. Open source (MIT). www.sandstre.am/kit
  • sandstre.am — den här sidan: eget designsystem, plain static och Claude Code som primärt verktyg. Inget build-steg, deployad via GitHub Pages.
  • Synapse — LLM-agnostisk kodningsagent i ren bash: keyword-router, dual-mode tool-calling över sju leverantörer och ett verktygsbibliotek bakom scope- och kommando-gate. Open source (MIT). github.com/sandstream/synapse
